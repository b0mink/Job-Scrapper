Job Scrapper 
=============
A job extracting web scraper from Indeed.com with **GoLang** & **HTML**  \
https://github.com/b0mink/Job-Scrapper


How to Use 
---------------
1. Enter a keyword in the search box
2. Click 'GENERATE' button
3. 'jobs.csv' will be downloaded
